# 数理小精灵：十之秘境

一款针对5-7岁儿童的数学学习RPG游戏，通过宠物对战的方式练习10以内的加减法。

## 游戏特色

- **宠物养成系统**：选择初始宠物，通过战斗获得经验值升级进化
- **回合制战斗**：正确回答数学题来攻击敌人
- **难度递进**：三个区域逐步提升难度
- **儿童友好设计**：大按钮、明亮色彩、正向反馈

## 如何运行

1. 直接用浏览器打开 `index.html` 文件
2. 选择你的初始宠物
3. 点击"探索"按钮遭遇野怪
4. 回答数学题来攻击敌人
5. 击败敌人获得经验值，提升宠物等级

## 数学规则

- **加法**：A + B = C，其中 A ≥ 0, B ≥ 0, C ≤ 10
- **减法**：X - Y = Z，其中 X ≤ 10, Y ≥ 0, X ≥ Y

## 关卡设计

1. **新手村（1-3级）**：只有结果 ≤ 5 的加法
2. **微风平原（4-6级）**：引入 ≤ 5 的减法，以及结果 6-8 的加法
3. **巨石山谷（7-9级）**：全面覆盖 ≤ 10 的加法和减法

## 技术实现

- 纯前端实现，无需服务器
- 使用HTML5 + CSS3 + JavaScript
- 响应式设计，支持平板和手机
- 状态机管理游戏流程
